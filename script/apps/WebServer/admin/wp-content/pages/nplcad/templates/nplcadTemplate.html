Build cube without parametres:

--Default radius={1,1,1},center={0,0,0},color={1,1,1};
cube();

Build cube with parametres:


--Set with limit parametres, the other will use default settings.
cube({radius={3, 3, 3},color={0,1,1},});	
--Use different radius for x, y, z position to creat cuboid.	
cube({radius={3, 4, 5},center={3, 3, 3}});

Sphere have five parametres:radius, center, color, slice and stacks. We rarely use the last two parametres.

--Default radius=1,center={0,0,0},color={1,1,1},slices = 16,stacks = 8.
sphere();	--build without parametres.
sphere({
  radius = 5,
  slices = 16,
  stacks = 8,
  center = {1,7,0},
  color = {160/255,134/255,132/255},
});

Cylinder also have five parametres: from, to, radius, color, slices. "from" parametre set where cylinder start and "to" means where cylinder end.


--Default from = {0,-1,0},to = {0,1,0},color={1,1,1},radius = 1,slices = 16.
cylinder();	--build without parametres.
cylinder({
  from = {5,-1,0},
  to = {5,1,0},
  radius = 1,
  slices = 16,
  color = {1,1,1},
});
Boolean operations consist of union, difference, intersection, which means OR, XOR, AND, respectively. See Boolean operations on Wikipedia.
union: 

union();
	cylinder({ radius = 0.7, from = {-1,0,0}, to = {1,0,0}, color = {0,1,0}, });
	cylinder({ radius = 0.7, from = {0,-1,0}, to = {0,1,0}, color = {0,1,0}, });
difference: 

difference() 
	color(0,0.5,1);
	cube({radius=3,});
	sphere({ radius = 4, color = {0,0,1}, });
intersection: 

intersection()
	sphere({ radius= 2.6, color = {1,0,1}, });
	cube({radius =2});

Use push(), pop() to build a children node. pop and push are explicitly named nodes, which can be used to specify the range of commands within.
